Tableau Dashboards: AirBnb Dataset

Summary: This project analyzes an Airbnb dataset to uncover trends in pricing, occupancy, and guest preferences across various locations. I chose this dataset to demonstrate my ability to analyze market dynamics and provide insights into the sharing economy.

Data source: link to dataset

Objective:: Using Tableau to create visually engaging and interactive dashboards that help stakeholders understand complex datasets at a glance. By leveraging Tableau's powerful data visualization capabilities, they can uncover insights, trends, and patterns that might be difficult to identify through raw data alone. Additionally, Tableau enabled me to share my findings effectively with non-technical audiences, facilitating data-driven decision-making across the organization.

Process: To create visualizations in Tableau, I connected to the data source and performed the necessary data preparation steps, such as filtering, grouping, or calculating new fields. Next, I utilized Tableau's drag-and-drop interface to build various charts and graphs, selecting appropriate visual elements to represent the data effectively. Finally, I combined these visualizations into an interactive dashboard, which can be shared with stakeholders to facilitate data-driven insights and decision-making.

Results: One-bedroom houses are the most popular choice on Airbnb, while listings with six or more bedrooms command the highest average prices. Zip code 98119 features the most expensive listings overall, and there are distinct seasonal trends where the average room prices decline.
